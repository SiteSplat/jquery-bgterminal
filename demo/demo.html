jQuery Background Terminal

Create a terminal-like animated background!

jQuery Background Terminal allows you to add a geeky-effect to your site. The plugin creates a terminal-like effect on an element, and automatically emulates the typing.

How to use

First off, you need to include jQuery, the javascript plugin, and the CSS in the head of your page: <script type="text/javascript" src="//code.jquery.com/jquery-1.9.1.js"></script>
<script type="text/javascript" src="jquery.bgterminal.min.js"></script>
<link href='bgterminal.css' rel='stylesheet' type='text/css' />

Once you did that, you can apply the effect to any element. However, it would be better to create a div as the first element the body, and apply the effect to it.

Also, note that by default - if you don't change the CSS - the terminal element will have position set to fixed. To avoid issues with the content, you can create an additional div with position relative and put your html there. <body>
<div id="terminal"></div>
<div id="content" style="position:relative">
  <!-- Your content here -->
</div>

Then, add the effect! <script>
  $(document).ready(function() {
    $('#terminal').bgTerminal({
      url: 'index.html',
      speed: 500,
      wait: 1000,
      glowing: true
    });
  });
</script>

Parameters

The following parameters are supported when calling bgTerminal:

namedescriptiondefault
url The page to load and display in the terminal undefined
speed Type speed in milliseconds. This is the time between two character are typed 200
wait The number of milliseconds to wait at the end of the line (when \n is found) 1000
glowing If true, the terminal will have a glowing effect false

Note: you can pass a callback function as a second parameter of bgTerminal. This function will be called once the terminal has been created

Change speed

A few functions can be called to retrieve or change the speed of the terminal, once it has been created. To call a function, you need to retrieve the bgterminal data from the element, and then call the methods: var terminal = $('#bgterminal').data('bgterminal');
terminal.speed(); // return an object containing the values of speed and wait
terminal.speedUp(amount); // speed both speed and wait time up 'amount' times (default: 1.5)
terminal.speedDown(amount); // speed both speed and wait time down 'amount' times (default: 1.5)
